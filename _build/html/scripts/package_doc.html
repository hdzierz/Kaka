scripts.configuration_parser.config_write(d)¶

Takes a dictionary with python object values and builds from it a yaml.safe_dump() friendly dictionary with just primitive types. In particular, datetime objects get represented as strings of format “dt(%Y-%m-%d)” or “dt(%Y-%m-%dT%H:%M:%SZ)”, and any strings with ‘#’ characters get represented as quoted strings

to prevent yaml from mistaking them for comments
Parameters:d – Dictionary to build a yaml.safe_dump() friendly representation of
Returns:yaml.safe_dump() friendly dictionary
scripts.configuration_parser.datetime_parse(d)¶

(Method to be used as the object_hook kwarg in json.load)

Returns a copy of the given dictionary with the string values for dates replaced with datetime objects, provided they match the formats “dt(%Y-%m-%d)” or “dt(%Y-%m-%dT%H:%M:%SZ)” :param d: Dictionary read from parsed config file :return: Copy of d with datetime objects replacing strings of dates
