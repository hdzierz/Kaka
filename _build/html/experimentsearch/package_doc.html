experimentsearch.views.stream_experiment_csv(request, experi_id)¶

Queries the genotype collection with the experiment that matches experi_id as a filter on ‘study’. Creates a csv representation of the query set.

Writes a http response which downloads the csv file for the client. This response is stored by this module to be downloaded by the download_experiment() view, which gets called by the index.html template once there is a redirect to the index() view, which this view returns
