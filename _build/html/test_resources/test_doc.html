Tests that the modules load_from_config and configuration_parser correctly read config files for loading data in a directory to the database, and then correctly loads them into the database

Ensures the load_from_config script itterates through the test resources folder instead of the default data directory

tearDown()¶

Clears the test database and resets the test configuration files

Tests that an error gets raised by a ConfigParser when a config file has no date in a date field

Tests that an error gets raised by a ConfigParser when a config file has a badly formatted date in a date field

test_config_parser_to_json_yaml()¶

test_does_not_run_twice()¶

Tests that load_from_config.run() marks all the config files it comes across as loaded and does not load from their directories when called a second time
